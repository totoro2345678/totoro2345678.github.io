Keyphrase Search

一、介紹

這是一個chrome extension ，透過選取字，按右鍵即可得到該字在維基百科上的定義

二、 起源

為了讓大家在讀論文或是專業文章時，能夠方便的查詢一些專有名詞的解釋，因此製作了 wiki chrome extension ，讓大家在chrome 瀏覽器裡，能夠快速且方便的向 wikipedia 查詢詞的定義，在這裡附上下載連結

三、 現有相關產品概況及比較

目前在chrome 商店上的chrome extension大多是單純的「字翻譯」而已，並非是得到「詞的解釋」，大多時候必須開分頁查詢詞的解釋，如果是需要大量查詢詞的解釋的使用者，應該要花掉許多時間成本，在開分頁瀏覽文章，找尋詞的解釋；因此我開發了專門查詢詞的解釋的chrome extension，讓大家能夠在找時詞的解釋時，能夠更便利。

四、設計方法、流程

將使用者所選取的字先使用 wikipedia 官方 Opensearch API ，查詢與wikipedia title 最相關的字，在將得到的最相關的字，用 wikipedia summary api ，得到相關字的定義或解釋。

當使用選取詞在wikipedia有相關解釋時，除了會顯示該詞的定義外，如果想要更多資訊，在最下面有附上wikipedia 該詞的連結，可以前往查看；當使用者選取詞在wikipedia沒有相關解釋時，會跳出wikipedia not found，下面有一個連結會導回 Gooogle 搜尋。

提供英文版和中文版，英文版是原文，就是向wikipedia查詢的結果，中文版是英文版透過Google Translate翻譯而來的，目前只支援英文字的選取。
